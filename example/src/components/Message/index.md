## Message

提示组件

## API

| 属性      | 描述                     | 类型                                  | 默认值 | 必填 |
| --------- | ------------------------ | ------------------------------------- | ------ | ---- |
| className | -                        | string                                | -      | -    |
| style     | -                        | any                                   | -      | -    |
| content   | 提示内容                 | any                                   | -      | ✓    |
| type      | 提示类型，对应不同的样式 | "check" \| "fail" \| "lock" \| "info" | -      | ✓    |
| icon      | 自定义图标               | any                                   | -      | -    |
