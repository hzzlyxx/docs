## Message Props

> 提示组件

| 属性      | 描述                     | 类型                                  | 默认值 | 必填 |
| --------- | ------------------------ | ------------------------------------- | ------ | ---- |
| className | -                        | string                                | -      | -    |
| style     | -                        | CSSProperties                         | -      | -    |
| content   | 提示内容                 | ReactNode                             | -      | ✓    |
| type      | 提示类型，对应不同的样式 | "check" \| "fail" \| "lock" \| "info" | -      | ✓    |
| icon      | 自定义图标               | ReactNode                             | -      | -    |
