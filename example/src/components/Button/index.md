## Button Props

> 按钮组件

| 属性     | 描述             | 类型                                                  | 默认值 | 必填 |
| -------- | ---------------- | ----------------------------------------------------- | ------ | ---- |
| htmlType | Button 类型      | "button" \| "submit" \| "reset"                       | -      | -    |
| href     | Button 跳转地址  | string                                                | -      | -    |
| loading  | 是否显示 loading | boolean                                               | false  | -    |
| disabled | 是否禁用         | boolean                                               | -      | -    |
| onClick  | Button 点击事件  | (event?: MouseEvent<HTMLElement, MouseEvent>) => void | -      | -    |
